# Spiff Sans

The default font for [Spiff.scad](https://github.com/stuartpb/Spiff.scad).

## Building

Using yaml2scad.js from [spaggers](https://github.com/stuartpb/spaggers):

    node yaml2scad.js spiffsans.yaml spiffsans.scad
